# Network Packet Analyzer

A Python-based network packet analyzer built using Scapy for capturing and inspecting network traffic in real time. This project demonstrates the fundamentals of packet sniffing, network protocols, and traffic analysis by extracting and displaying key packet information such as source and destination IP addresses, protocol types, port numbers, and payload data.

## Features

* Real-time packet capture
* IP address extraction
* TCP, UDP, and ICMP protocol identification
* Source and destination port analysis
* Payload inspection
* Traffic filtering support
* Educational demonstration of packet structures and network communication

## Technologies Used

* Python
* Scapy
* Npcap (Windows packet capture driver)

## Learning Objectives

This project was developed to explore:

* Network packet structures
* OSI and TCP/IP models
* Protocol analysis
* Traffic monitoring techniques
* Cybersecurity and network forensics fundamentals

## Disclaimer

This tool is intended for educational purposes and authorized network analysis only. Users are responsible for complying with applicable laws, regulations, and organizational policies when capturing or analyzing network traffic.
